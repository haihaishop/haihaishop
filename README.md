
# haihaishop

-    SpringMVC
-    MyBatis
-    BootStrap

## 数据库：
-    MySQL

## 开发语言：
-    Java
-    Jsp
-    JavaScript

## 应用技术：
-    JQuery
-    Ajax
-    Maven

## 开发环境：
-    Intellij idea
-    Redis
-    ActiveMQ
-    MySQL
-    Tomcat

## 运行配置：
  直接下载或者 `git clone https://github.com/haihaishop/haihaishop.git`</br>
  下载完成后打开haihaishop/src/main/resources</br>
  修改以下内容：
  - db.properties
  -- 修改用户名、密码、数据库名称
  - redis.properties</br>
  -- 修改用户名、密码
  - 打开mysql， 执行tables.sql中的语句，即可完成建表。</br>
